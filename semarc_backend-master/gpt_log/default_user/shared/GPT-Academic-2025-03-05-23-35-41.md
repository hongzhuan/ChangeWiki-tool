#### 功能概述
该模块是一个用于 Sphinx 文档生成的插件，提供了一个处理手册页的角色 (`man`) 的功能，使得用户能够链接到 Linux 手册页。

#### 关键功能
1. **链接节点创建 (make_link_node)**:
   - 输入手册页名称 (`name`) 和页码 (`manpage_num`)，生成一个指向指定手册页的链接节点。
   - 如果没有提供 URL 模板，则使用默认的手册页 URL 格式。

2. **手册页角色处理 (man_role)**:
   - 解析输入文本格式 `name(num)`，提取手册页名称和页码。
   - 调用 `make_link_node` 生成手册链接节点。

3. **插件设置 (setup)**:
   - 定义并注册 `man` 角色到 Sphinx 应用中。
   - 提供配置项 `man_url_regex`，允许用户自定义手册页的 URL 格式。

#### 使用说明
- 用户可以在 Sphinx 文档中使用 `:man:name(num):` 语法插入手册页链接。
- 该插件延展了 Sphinx 以更好地支持 Linux 手册页的引用和文档化。 

#### 注意
- 需要安装 `docutils` 库以支持节点和角色的创建与管理。
